# 2. 제어 흐름

## 1. 식(Expression)으로서의 제어문

코틀린의 `if`와 `when`은 값을 반환할 수 있는 식(Expression)입니다. 자바의 삼항 연산자(`condition ? true : false`)를 대체하며, 변수에 할당하거나 함수의 반환값으로 사용할 수 있습니다.

```kotlin
val max = if (a > b) a else b

val result = when (x) {
    1 -> "One"
    2 -> "Two"
    else -> "Other"
}
```

## 2. 실무 압축 `for`문 활용 (3가지 필수 관용구)

### 단순 아이템 순회

가장 기본적인 형태입니다. 리스트나 배열의 요소를 하나씩 꺼내옵니다.

```kotlin
for (item in items) {
    println(item)
}
```

### 인덱스와 함께 순회

리사이클러뷰 어댑터 등에서 인덱스가 필요할 때 유용합니다. `withIndex()` 함수를 사용하면 인덱스와 값을 동시에 얻을 수 있습니다.

```kotlin
for ((index, item) in items.withIndex()) {
    println("Index: $index, Item: $item")
}
```

### 단순 반복 작업

특정 횟수만큼 반복해야 할 때 `repeat()` 함수를 사용합니다. 람다식 내부에서 현재 반복 횟수(0부터 시작)를 인자로 받을 수 있습니다.

```kotlin
repeat(5) { count ->
    println("Hello $count") // 0, 1, 2, 3, 4 출력
}
```

## 3. 범위(Range)와 진행(Progression)

`..` 연산자를 사용하여 범위를 표현할 수 있으며, `step`이나 `downTo`를 사용하여 진행 방향과 간격을 조절할 수 있습니다.

```kotlin
for (i in 1..10 step 2) { // 1, 3, 5, 7, 9
    print(i)
}
```

## 4. 예외 처리 (Exceptions)

코틀린의 `try-catch` 블록도 값을 반환하는 식(Expression)입니다. 따라서 변수에 할당하거나 함수의 반환값으로 사용할 수 있습니다.

```kotlin
val number = try {
    Integer.parseInt(input)
} catch (e: NumberFormatException) {
    null
}
```
